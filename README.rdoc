== Resugen

An attempt to hold user content and attempt to generate focused resumes based on simple templates.

== Usage

This application will be hosted somewhere in the future.

Prequisites:

- Ruby 1.9
- RubyGems
- Bundler

Run Application:

1. Clone Repo
2. bundle install
3. rails s
